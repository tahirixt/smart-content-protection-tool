A simple plugin to disable text selection and prevent image dragging on your WordPress site.

== Description ==

The "Disable Text Selection, Right Click and Image Dragging" plugin provides a simple way to enhance the user experience by preventing unintended text selection and image dragging. While this can deter casual copying or dragging, always remember that content displayed on the client-side can be accessed by those with the right knowledge.

= Features =

* Disable Source
* Disable Copy Content
* Disable Shortcuts (Ctrl + C, Ctrl + U and more...)
* Disable Developer Tool.
* Disable text selection.
* Prevent image dragging.
* Lightweight with minimal impact on performance.

== Installation ==

1. Upload `content-protection` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Once activated, the functionalities will be applied site-wide.

== Frequently Asked Questions ==

= Can users still copy my content using other methods? =

Yes, determined users can always find ways to copy content or bypass these restrictions. This plugin provides a deterrent, but always assume that anything on the frontend is accessible.

= Does this impact SEO or accessibility? =

No, this plugin should not impact your site's SEO. However, be mindful that disabling certain functionalities might affect the user experience for some visitors.
